🛠️ How to use the manager

Once the app is open, follow these steps to manage your game files:

  1. Select the game folder from the main menu inside the app.
  2. Drag and drop your mod folders into the application interface.
  3. Press the Install button to apply the changes.
  4. Use the toggle switch to enable or disable mods as needed.

⚖️ Managing Conflicts

When you install many mods, some might change the same part of the game. Gloss Mod Manager identifies these conflicts for you. When the app finds a conflict, it prompts you to choose which mod to use. You can change your choices at any time through the same menu.

📂 Why use this instead of other tools?

Many mod managers are difficult to understand. This tool stays lightweight. It does not run in the background while you play. It only changes files when you tell it to. This keeps your computer running fast and your games stable.
